#!/bin/bash

# Bucle infinito para solicitar continuamente la palabra clave
while true; do
  echo "Introduce la palabra clave:"
  read palabra_clave

  # Comprobar si la palabra clave es "secreto"
  if [ "$palabra_clave" == "secreto" ]; then
    echo "¡Bienvenido! Has ingresado la palabra clave correcta."
    break # Salir del bucle
  else
    echo "Palabra clave incorrecta. Intenta de nuevo."
  fi
done
Este script utiliza un bucle while para solicitar continuamente la palabra clave al usuario utilizando el comando read en Bash. Si la palabra clave ingresada es "secreto", el script mostrará un mensaje de bienvenida y saldrá del bucle utilizando la declaración break. Si la palabra clave ingresada es incorrecta, el script mostrará un mensaje de error y volverá a solicitar la palabra clave. Puedes guardar este script en un archivo con el nombre "clave" y darle permisos de ejecución para que puedas ejecutarlo en tu terminal de Ubuntu. Ten en cuenta que este es un ejemplo simple y puede ser mejorado para adaptarse a tus necesidades específicas.
